Privacy Policy

Fuel Prices App (de.tankstellen.app) — Last updated: 7 April 2026

1. Overview

Fuel Prices ("the app") is a free, open-source fuel and EV charging price comparison app. It is designed with a local-first, privacy-respecting architecture. There are no ads, no analytics, and no tracking.

2. Data collected

2.1 Location data (approximate)

When you grant location permission, the app reads your approximate position to find nearby fuel stations. Your coordinates are sent to third-party price APIs (see Section 4) as part of the search query. Location data is not stored on any server we operate and is never used for tracking or profiling.

2.2 API key

You provide your own API key (e.g. Tankerkoenig). The key is stored locally in encrypted secure storage (Android Keystore / iOS Keychain). It is sent only to the corresponding API provider and is never transmitted to us or any other party.

2.3 Favorites, profiles, and settings

Your favorites, search profiles, fuel preferences, and app settings are stored locally on your device using Hive (an encrypted local database).

2.4 TankSync (optional cloud sync)

If you opt in to TankSync, an anonymous account is created via Supabase. The following data is synced to the server:

  • Anonymous user ID (UUID, no email or name)
  • Favorite station IDs
  • Price alert configurations
  • Community price reports (station ID, fuel type, price, timestamp)

TankSync is entirely optional and disabled by default. You can view, export, and delete all server-side data from the "Data Transparency" screen in the app.

3. Data NOT collected

The app does not collect:

  • Name, email address, or phone number
  • Financial or payment information
  • Health or fitness data
  • Contacts, messages, or call logs
  • Photos, videos, or files
  • Browsing history
  • Device advertising identifiers
  • Analytics or crash reports (unless Sentry is opted into — not currently active)

4. Third-party services

The app communicates with the following third-party APIs:

  • Tankerkoenig (creativecommons.tankerkoenig.de) — German fuel prices. Receives: search coordinates, API key.
  • Prix Carburants (data.economie.gouv.fr) — French fuel prices. Receives: search coordinates.
  • Italian fuel price API (osservaprezzi.mise.gov.it) — Italian fuel prices. Receives: search coordinates.
  • Spanish fuel price API (sedeaplicaciones.minetur.gob.es) — Spanish fuel prices. Receives: search coordinates.
  • Austrian fuel price API (spritpreisrechner.at) — Austrian fuel prices. Receives: search coordinates.
  • Belgian fuel price API (fuelwatch.be) — Belgian fuel prices. Receives: search coordinates.
  • Luxembourg fuel price API (data.public.lu) — Luxembourg fuel prices. Receives: search coordinates.
  • OpenChargeMap (api.openchargemap.io) — EV charging station data. Receives: search coordinates.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — Geocoding. Receives: search text or coordinates.
  • OpenStreetMap tile servers (tile.openstreetmap.org) — Map tiles. Receives: tile coordinates.
  • Supabase (only if TankSync is enabled) — Cloud sync backend. Receives: anonymous user data (see Section 2.4).

Each provider has its own privacy policy. We encourage you to review them. The app does not share data between these providers.

5. Data security

  • All network communication uses HTTPS (TLS encryption in transit).
  • API keys are stored in platform-native encrypted storage (Android Keystore).
  • Local data is stored in Hive (local database on device storage).
  • No data is sent to any server unless you initiate a search or enable TankSync.

6. Your rights

You have the right to:

  • Access — View all locally stored data in the app's Storage section.
  • Export — Export your TankSync data as JSON from the Data Transparency screen.
  • Delete — Delete all local data via Settings > Delete all data. Delete all server data via TankSync > Data Transparency > Delete everything.
  • Withdraw consent — Revoke location permission in your device settings at any time. Disable TankSync at any time.

7. Children's privacy

The app is not directed at children under 13. We do not knowingly collect personal information from children.

8. Changes to this policy

We may update this policy from time to time. Changes will be published at this URL and noted with an updated date. Continued use of the app constitutes acceptance of the updated policy.
