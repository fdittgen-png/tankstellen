Dostępne języki: English Deutsch Français Español Italiano Nederlands Português Svenska Suomi Dansk Polski Slovenščina 한국어

1. Omówienie

Sparkilo (wcześniej Fuel Prices Europe & More) to bezpłatna aplikacja open source do porównywania cen paliw i punktów ładowania pojazdów elektrycznych. Zaprojektowano ją w architekturze local-first, szanującej prywatność. Bez reklam, bez pikseli śledzących, bez identyfikatorów reklamowych.

2. Wykorzystywane dane

2.1 Przybliżona lokalizacja

Po przyznaniu uprawnienia do lokalizacji aplikacja odczytuje Twoją przybliżoną pozycję, by znaleźć pobliskie stacje paliw i punkty ładowania. Współrzędne są wysyłane do interfejsów API stron trzecich (zob. sekcja 4) jako część zapytania. Twoja lokalizacja nie jest przechowywana na żadnym z naszych serwerów i nigdy nie służy do śledzenia ani profilowania.

2.2 Klucze API podawane przez Ciebie

Jeśli podasz własny klucz API (np. Tankerkönig), zostanie on zapisany lokalnie w szyfrowanym magazynie (Android Keystore / iOS Keychain). Klucz wysyłany jest wyłącznie do odpowiedniego dostawcy API — nigdy do nas ani do osób trzecich.

2.3 Ulubione, profile i ustawienia

Twoje ulubione, profile wyszukiwania, preferencje paliw i ustawienia aplikacji przechowywane są lokalnie na urządzeniu w bazie Hive.

2.4 TankSync (opcjonalna synchronizacja w chmurze)

Po włączeniu TankSync tworzone jest anonimowe konto w Supabase. Synchronizowane są:

  • Anonimowy identyfikator użytkownika (UUID — bez e-maila, bez imienia)
  • Identyfikatory ulubionych stacji
  • Konfiguracje alertów cenowych
  • Społecznościowe zgłoszenia cen (ID stacji, rodzaj paliwa, cena, znacznik czasu)

TankSync jest opcjonalny i domyślnie wyłączony. Możesz przeglądać, eksportować i usuwać wszystkie dane po stronie serwera z ekranu « Przejrzystość danych » w aplikacji.

2.5 Raporty awarii i diagnostyki (opt-in)

Po włączeniu diagnostyki w Ustawieniach do Sentry wysyłane są anonimowe raporty awarii i ślady wydajności. Nie są przekazywane żadne dane osobowe, lokalizacja ani treści. Diagnostyka jest domyślnie wyłączona.

3. Dane, których NIE zbieramy

  • Imię i nazwisko, adres e-mail lub numer telefonu
  • Dane finansowe lub płatnicze
  • Dane zdrowotne lub fitness
  • Kontakty, wiadomości lub rejestry połączeń
  • Zdjęcia, filmy lub pliki (dostęp do aparatu i biblioteki zdjęć jest używany tylko do skanowania paragonów, gdy Ty go uruchamiasz)
  • Historia przeglądania
  • Identyfikatory reklamowe urządzenia

4. Usługi stron trzecich

Aplikacja komunikuje się z następującymi interfejsami API stron trzecich:

  • Tankerkönig (creativecommons.tankerkoenig.de) — ceny niemieckie. Otrzymuje: współrzędne wyszukiwania, Twój klucz API.
  • Prix Carburants (data.economie.gouv.fr) — ceny francuskie. Otrzymuje: współrzędne wyszukiwania.
  • Włoskie API cen (osservaprezzi.mise.gov.it) — ceny włoskie. Otrzymuje: współrzędne wyszukiwania.
  • Hiszpańskie API cen (sedeaplicaciones.minetur.gob.es) — ceny hiszpańskie. Otrzymuje: współrzędne wyszukiwania.
  • Spritpreisrechner (spritpreisrechner.at) — ceny austriackie. Otrzymuje: współrzędne wyszukiwania.
  • FuelWatch (fuelwatch.be) — ceny belgijskie. Otrzymuje: współrzędne wyszukiwania.
  • data.public.lu — ceny luksemburskie. Otrzymuje: współrzędne wyszukiwania.
  • OpenChargeMap (api.openchargemap.io) — punkty ładowania. Otrzymuje: współrzędne wyszukiwania.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — geokodowanie. Otrzymuje: tekst lub współrzędne wyszukiwania.
  • Serwery kafelków OpenStreetMap (tile.openstreetmap.org) — kafelki mapy. Otrzymuje: współrzędne kafelka.
  • Supabase (tylko gdy TankSync jest włączony) — backend synchronizacji w chmurze.
  • Sentry (tylko gdy diagnostyka jest włączona) — anonimowe raporty awarii i wydajności.

Każdy dostawca ma własną politykę prywatności. Zachęcamy do zapoznania się z nimi. Aplikacja nie udostępnia danych pomiędzy tymi dostawcami.

5. Bezpieczeństwo danych

  • Cała komunikacja sieciowa odbywa się przez HTTPS (szyfrowanie TLS w transporcie).
  • Klucze API są przechowywane w natywnym, szyfrowanym magazynie platformy (Android Keystore / iOS Keychain).
  • Dane lokalne są przechowywane na Twoim urządzeniu w bazie Hive.
  • Żadne dane nie są wysyłane do serwerów, dopóki nie uruchomisz wyszukiwania ani nie włączysz TankSync / diagnostyki.

6. Twoje prawa

Masz prawo do:

  • Dostępu — przeglądania wszystkich lokalnie zapisanych danych w sekcji Pamięć w aplikacji.
  • Eksportu — eksportu danych TankSync w formacie JSON z ekranu Przejrzystość danych.
  • Usunięcia — usunięcia wszystkich danych lokalnych przez Ustawienia → Usuń wszystkie dane; usunięcia wszystkich danych po stronie serwera przez TankSync → Przejrzystość danych → Usuń wszystko.
  • Wycofania zgody — odebranie uprawnienia do lokalizacji w ustawieniach urządzenia lub wyłączenie TankSync / diagnostyki w dowolnej chwili.

7. Prywatność dzieci

Aplikacja nie jest skierowana do dzieci poniżej 13 roku życia. Świadomie nie zbieramy danych osobowych od dzieci.

8. Zmiany w niniejszej polityce

Co pewien czas możemy aktualizować tę politykę. Zmiany będą publikowane pod tym adresem URL z aktualną datą. Dalsze korzystanie z aplikacji oznacza akceptację zaktualizowanej polityki.
