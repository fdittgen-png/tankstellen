Política de privacidade

Idiomas disponíveis: English Deutsch Français Español Italiano Nederlands Português Svenska Suomi Dansk Polski Slovenščina 한국어

1. Visão geral

O Sparkilo (anteriormente Fuel Prices Europe & More) é uma aplicação gratuita e de código aberto para comparar preços de combustível e de pontos de carregamento elétrico. É construída segundo uma arquitetura local-first, respeitadora da privacidade. Sem anúncios, sem píxeis de rastreamento, sem identificadores publicitários.

2. Dados que utilizamos

2.1 Localização aproximada

Quando concede a permissão de localização, a app lê a sua posição aproximada para encontrar postos e pontos de carregamento próximos. As coordenadas são enviadas a APIs de terceiros (ver secção 4) como parte da consulta. A sua localização não é armazenada em nenhum servidor que operamos e nunca é usada para rastreamento ou perfis.

2.2 Chaves de API que você fornece

Se fornecer a sua própria chave de API (por exemplo, Tankerkönig), ela é guardada localmente em armazenamento cifrado (Android Keystore / iOS Keychain). A chave é enviada apenas ao fornecedor de API correspondente, nunca a nós nem a terceiros.

2.3 Favoritos, perfis e definições

Os seus favoritos, perfis de pesquisa, preferências de combustível e definições da app são guardados localmente no seu dispositivo através do Hive, uma base de dados local incorporada.

2.4 TankSync (sincronização na nuvem opcional)

Se ativar o TankSync, é criada uma conta anónima através do Supabase. Os dados sincronizados são:

  • Identificador anónimo de utilizador (UUID — sem email nem nome)
  • IDs dos postos favoritos
  • Configurações de alertas de preço
  • Relatos comunitários de preço (ID do posto, tipo de combustível, preço, carimbo de tempo)

O TankSync é opcional e desativado por predefinição. Pode ver, exportar e eliminar todos os dados do servidor a partir do ecrã « Transparência de dados » dentro da app.

2.5 Relatórios de erro e diagnóstico (opt-in)

Se ativar os diagnósticos nas Definições, são enviados ao Sentry relatórios anónimos de erro e traços de desempenho. Não é incluída nenhuma informação pessoal, localização nem conteúdo. Os diagnósticos estão desativados por predefinição.

3. Dados que NÃO recolhemos

  • Nome, endereço de email ou número de telefone
  • Informações financeiras ou de pagamento
  • Dados de saúde ou de fitness
  • Contactos, mensagens ou registos de chamadas
  • Fotos, vídeos ou ficheiros (o acesso à câmara e à galeria é usado apenas para digitalizar talões quando você inicia)
  • Histórico de navegação
  • Identificadores publicitários do dispositivo

4. Serviços de terceiros

A app comunica com as seguintes APIs de terceiros:

  • Tankerkönig (creativecommons.tankerkoenig.de) — preços alemães. Recebe: coordenadas de pesquisa, a sua chave de API.
  • Prix Carburants (data.economie.gouv.fr) — preços franceses. Recebe: coordenadas de pesquisa.
  • API italiana de preços (osservaprezzi.mise.gov.it) — preços italianos. Recebe: coordenadas de pesquisa.
  • API espanhola de preços (sedeaplicaciones.minetur.gob.es) — preços espanhóis. Recebe: coordenadas de pesquisa.
  • Spritpreisrechner (spritpreisrechner.at) — preços austríacos. Recebe: coordenadas de pesquisa.
  • FuelWatch (fuelwatch.be) — preços belgas. Recebe: coordenadas de pesquisa.
  • data.public.lu — preços luxemburgueses. Recebe: coordenadas de pesquisa.
  • OpenChargeMap (api.openchargemap.io) — pontos de carregamento. Recebe: coordenadas de pesquisa.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — geocodificação. Recebe: texto ou coordenadas de pesquisa.
  • Servidores de tiles do OpenStreetMap (tile.openstreetmap.org) — tiles do mapa. Recebe: coordenadas do tile.
  • Supabase (apenas se o TankSync estiver ativado) — backend de sincronização na nuvem.
  • Sentry (apenas se os diagnósticos estiverem ativados) — relatórios anónimos de erro e desempenho.

Cada fornecedor tem a sua própria política de privacidade. Recomendamos que as consulte. A app não partilha dados entre estes fornecedores.

5. Segurança dos dados

  • Toda a comunicação em rede usa HTTPS (cifragem TLS em trânsito).
  • As chaves de API são guardadas no armazenamento cifrado nativo da plataforma (Android Keystore / iOS Keychain).
  • Os dados locais são guardados no seu dispositivo através do Hive.
  • Não são enviados dados para qualquer servidor a menos que inicie uma pesquisa ou ative o TankSync / diagnósticos.

6. Os seus direitos

Tem o direito de:

  • Acesso — ver todos os dados guardados localmente na secção Armazenamento da app.
  • Exportação — exportar os seus dados TankSync em JSON a partir do ecrã Transparência de dados.
  • Eliminação — eliminar todos os dados locais via Definições → Eliminar todos os dados; eliminar todos os dados de servidor via TankSync → Transparência de dados → Eliminar tudo.
  • Retirar consentimento — revogar a permissão de localização nas definições do dispositivo, ou desativar o TankSync / diagnósticos a qualquer momento.

7. Privacidade de crianças

A app não é dirigida a crianças com menos de 13 anos. Não recolhemos conscientemente informação pessoal de crianças.

8. Alterações a esta política

Podemos atualizar esta política periodicamente. As alterações serão publicadas neste URL com uma data atualizada. A continuação do uso da app constitui aceitação da política atualizada.
