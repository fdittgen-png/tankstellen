1. Panoramica

Sparkilo (precedentemente Fuel Prices Europe & More) è un'app gratuita e open source per il confronto dei prezzi del carburante e delle stazioni di ricarica. È costruita secondo un'architettura local-first e rispettosa della privacy. Niente pubblicità, niente pixel di tracciamento, niente identificatori pubblicitari.

2. Dati utilizzati

2.1 Posizione approssimativa

Quando concedi l'autorizzazione alla posizione, l'app legge la tua posizione approssimativa per trovare distributori e punti di ricarica nelle vicinanze. Le coordinate vengono inviate ad API di terze parti (vedi sezione 4) come parte della ricerca. La tua posizione non viene memorizzata su alcun server da noi gestito e non viene mai usata per tracciamento o profilazione.

2.2 Chiavi API che fornisci tu

Se fornisci una tua chiave API (es. Tankerkönig), viene memorizzata localmente in uno storage cifrato (Android Keystore / iOS Keychain). La chiave viene inviata solo al fornitore di API corrispondente, mai a noi o a terzi.

2.3 Preferiti, profili e impostazioni

Preferiti, profili di ricerca, preferenze di carburante e impostazioni dell'app sono memorizzati localmente sul tuo dispositivo tramite Hive, un database locale incorporato.

2.4 TankSync (sincronizzazione cloud opzionale)

Se attivi TankSync, viene creato un account anonimo tramite Supabase. I dati sincronizzati sono:

  • Identificatore utente anonimo (UUID — nessuna email, nessun nome)
  • ID dei distributori preferiti
  • Configurazioni degli avvisi di prezzo
  • Segnalazioni di prezzo dalla community (ID distributore, tipo carburante, prezzo, timestamp)

TankSync è opzionale e disattivato per impostazione predefinita. Puoi visualizzare, esportare ed eliminare tutti i dati lato server dalla schermata « Trasparenza dei dati » all'interno dell'app.

2.5 Rapporti di crash e diagnostica (opt-in)

Se attivi la diagnostica nelle Impostazioni, vengono inviati a Sentry rapporti anonimi di crash e tracce di performance. Non viene incluso alcun dato personale, posizione o contenuto. La diagnostica è disattivata per impostazione predefinita.

3. Dati che NON raccogliamo

  • Nome, indirizzo email o numero di telefono
  • Informazioni finanziarie o di pagamento
  • Dati di salute o fitness
  • Contatti, messaggi o registri delle chiamate
  • Foto, video o file (l'accesso a fotocamera e libreria foto è usato solo per scansionare scontrini quando lo avvii tu)
  • Cronologia di navigazione
  • Identificatori pubblicitari del dispositivo

4. Servizi di terze parti

L'app comunica con le seguenti API di terze parti:

  • Tankerkönig (creativecommons.tankerkoenig.de) — prezzi tedeschi. Riceve: coordinate di ricerca, la tua chiave API.
  • Prix Carburants (data.economie.gouv.fr) — prezzi francesi. Riceve: coordinate di ricerca.
  • API italiana dei prezzi (osservaprezzi.mise.gov.it) — prezzi italiani. Riceve: coordinate di ricerca.
  • API spagnola dei prezzi (sedeaplicaciones.minetur.gob.es) — prezzi spagnoli. Riceve: coordinate di ricerca.
  • Spritpreisrechner (spritpreisrechner.at) — prezzi austriaci. Riceve: coordinate di ricerca.
  • FuelWatch (fuelwatch.be) — prezzi belgi. Riceve: coordinate di ricerca.
  • data.public.lu — prezzi lussemburghesi. Riceve: coordinate di ricerca.
  • OpenChargeMap (api.openchargemap.io) — stazioni di ricarica EV. Riceve: coordinate di ricerca.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — geocodifica. Riceve: testo o coordinate di ricerca.
  • Server di tile OpenStreetMap (tile.openstreetmap.org) — tile della mappa. Riceve: coordinate del tile.
  • Supabase (solo se TankSync è attivo) — backend di sincronizzazione cloud.
  • Sentry (solo se la diagnostica è attiva) — rapporti anonimi di crash e performance.

Ogni fornitore ha la propria informativa sulla privacy. Ti invitiamo a leggerle. L'app non condivide dati tra questi fornitori.

5. Sicurezza dei dati

  • Tutte le comunicazioni di rete usano HTTPS (cifratura TLS in transito).
  • Le chiavi API sono memorizzate nello storage cifrato nativo della piattaforma (Android Keystore / iOS Keychain).
  • I dati locali vengono memorizzati sul tuo dispositivo tramite Hive.
  • Nessun dato viene inviato a server a meno che tu non avvii una ricerca o attivi TankSync / la diagnostica.

6. I tuoi diritti

Hai il diritto di:

  • Accesso — visualizzare tutti i dati memorizzati localmente nella sezione Archiviazione dell'app.
  • Esportazione — esportare i tuoi dati TankSync in JSON dalla schermata Trasparenza dei dati.
  • Cancellazione — eliminare tutti i dati locali tramite Impostazioni → Elimina tutti i dati; eliminare tutti i dati del server tramite TankSync → Trasparenza dei dati → Elimina tutto.
  • Revocare il consenso — revocare l'autorizzazione alla posizione nelle impostazioni del dispositivo o disattivare TankSync / la diagnostica in qualsiasi momento.

7. Privacy dei minori

L'app non è destinata a bambini sotto i 13 anni. Non raccogliamo consapevolmente informazioni personali dai minori.

8. Modifiche a questa informativa

Potremmo aggiornare periodicamente questa informativa. Le modifiche saranno pubblicate a questo URL con una data aggiornata. L'uso continuato dell'app costituisce accettazione della versione aggiornata.
