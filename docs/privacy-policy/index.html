Privacy Policy

Available languages: English Deutsch Français Español Italiano Nederlands Português Svenska Suomi Dansk Polski Slovenščina 한국어

1. Overview

Sparkilo (formerly known as Fuel Prices Europe & More) is a free, open-source fuel and EV charging price comparison app. It is built around a local-first, privacy-respecting architecture. There are no ads, no tracking pixels, and no advertising identifiers.

2. Data we use

2.1 Approximate location

When you grant location permission, the app reads your approximate position to find nearby fuel and charging stations. Coordinates are sent to third-party price APIs (see Section 4) as part of the search query. Your location is not stored on any server we operate and is never used for tracking or profiling.

2.2 API keys you provide

If you provide your own API key (e.g. for Tankerkönig), it is stored locally in encrypted secure storage (Android Keystore / iOS Keychain). The key is sent only to the corresponding API provider and never to us or any other party.

2.3 Favorites, profiles and settings

Your favorites, search profiles, fuel preferences and app settings are stored locally on your device using Hive, an embedded local database.

2.4 TankSync (optional cloud sync)

If you opt in to TankSync, an anonymous account is created via Supabase. The data synced is:

  • Anonymous user identifier (a UUID — no email or name)
  • Favorite station IDs
  • Price alert configurations
  • Community price reports (station ID, fuel type, price, timestamp)

TankSync is optional and disabled by default. You can view, export and delete all server-side data from the Data Transparency screen inside the app.

2.5 Crash and diagnostic reports (opt-in)

If you enable diagnostic reporting in Settings, anonymous crash reports and performance traces are sent to Sentry. No personally identifying information, location or content is included. Diagnostic reporting is off by default.

3. Data we do NOT collect

  • Name, email address or phone number
  • Financial or payment information
  • Health or fitness data
  • Contacts, messages or call logs
  • Photos, videos or files (camera and photo library access is only used for receipt scanning when you initiate it)
  • Browsing history
  • Device advertising identifiers

4. Third-party services

The app communicates with the following third-party APIs:

  • Tankerkönig (creativecommons.tankerkoenig.de) — German fuel prices. Receives: search coordinates, your API key.
  • Prix Carburants (data.economie.gouv.fr) — French fuel prices. Receives: search coordinates.
  • Italian fuel-price API (osservaprezzi.mise.gov.it) — Italian fuel prices. Receives: search coordinates.
  • Spanish fuel-price API (sedeaplicaciones.minetur.gob.es) — Spanish fuel prices. Receives: search coordinates.
  • Spritpreisrechner (spritpreisrechner.at) — Austrian fuel prices. Receives: search coordinates.
  • FuelWatch (fuelwatch.be) — Belgian fuel prices. Receives: search coordinates.
  • data.public.lu — Luxembourg fuel prices. Receives: search coordinates.
  • OpenChargeMap (api.openchargemap.io) — EV charging stations. Receives: search coordinates.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — geocoding. Receives: search text or coordinates.
  • OpenStreetMap tile servers (tile.openstreetmap.org) — map tiles. Receives: tile coordinates.
  • Supabase (only if TankSync is enabled) — cloud sync backend.
  • Sentry (only if diagnostic reporting is enabled) — anonymous crash and performance reports.

Each provider has its own privacy policy. We encourage you to review them. The app does not share data between these providers.

5. Data security

  • All network communication uses HTTPS (TLS in transit).
  • API keys are stored in platform-native encrypted storage (Android Keystore / iOS Keychain).
  • Local data is stored on your device using Hive.
  • No data is sent to any server unless you initiate a search or enable TankSync / diagnostics.

6. Your rights

You have the right to:

  • Access — view all locally stored data in the app's Storage section.
  • Export — export your TankSync data as JSON from the Data Transparency screen.
  • Delete — delete all local data via Settings → Delete all data; delete all server data via TankSync → Data Transparency → Delete everything.
  • Withdraw consent — revoke location permission in your device settings, or disable TankSync / diagnostics, at any time.

7. Children's privacy

The app is not directed at children under 13. We do not knowingly collect personal information from children.

8. Changes to this policy

We may update this policy from time to time. Changes will be published at this URL with an updated date. Continued use of the app constitutes acceptance of the updated policy.
