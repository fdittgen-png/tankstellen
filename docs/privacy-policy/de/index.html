1. Überblick

Sparkilo (vormals Fuel Prices Europe & More) ist eine kostenlose, quelloffene App zum Vergleich von Kraftstoff- und Ladestationspreisen. Die App basiert auf einer local-first, datenschutzfreundlichen Architektur. Es gibt keine Werbung, kein Tracking und keine Werbe-IDs.

2. Verwendete Daten

2.1 Ungefährer Standort

Wenn Sie den Standortzugriff erlauben, liest die App Ihre ungefähre Position aus, um nahegelegene Tankstellen und Ladesäulen zu finden. Die Koordinaten werden als Teil der Suchanfrage an Drittanbieter-APIs gesendet (siehe Abschnitt 4). Ihr Standort wird auf keinem von uns betriebenen Server gespeichert und niemals zum Tracking oder Profiling verwendet.

2.2 Eigene API-Schlüssel

Wenn Sie einen eigenen API-Schlüssel hinterlegen (z. B. für Tankerkönig), wird er lokal in einem verschlüsselten Speicher (Android Keystore / iOS Keychain) gehalten. Der Schlüssel wird ausschließlich an den jeweiligen API-Anbieter gesendet, niemals an uns oder Dritte.

2.3 Favoriten, Profile und Einstellungen

Ihre Favoriten, Suchprofile, Kraftstoffvorlieben und App-Einstellungen werden lokal auf Ihrem Gerät in Hive (eingebettete lokale Datenbank) gespeichert.

2.4 TankSync (optionale Cloud-Synchronisation)

Wenn Sie TankSync aktivieren, wird über Supabase ein anonymes Konto angelegt. Synchronisiert werden:

  • Anonyme Benutzer-ID (UUID — keine E-Mail-Adresse, kein Name)
  • Favoriten-Stationen-IDs
  • Konfigurationen für Preisalarme
  • Community-Preismeldungen (Stations-ID, Kraftstoffart, Preis, Zeitstempel)

TankSync ist optional und standardmäßig deaktiviert. Sie können sämtliche serverseitigen Daten im Bildschirm „Datentransparenz“ in der App einsehen, exportieren und löschen.

2.5 Absturz- und Diagnoseberichte (Opt-in)

Wenn Sie die Diagnoseberichte in den Einstellungen aktivieren, werden anonyme Absturzberichte und Performance-Traces an Sentry gesendet. Es werden keine personen­bezogenen Daten, Standortdaten oder Inhalte übermittelt. Diagnoseberichte sind standardmäßig deaktiviert.

3. Daten, die wir NICHT erheben

  • Name, E-Mail-Adresse oder Telefonnummer
  • Finanz- oder Zahlungsdaten
  • Gesundheits- oder Fitnessdaten
  • Kontakte, Nachrichten oder Anrufprotokolle
  • Fotos, Videos oder Dateien (Kamera- und Fotomediathek-Zugriff wird nur für das Scannen von Belegen genutzt, wenn Sie es selbst auslösen)
  • Browserverlauf
  • Werbe-IDs des Geräts

4. Drittanbieter-Dienste

Die App kommuniziert mit folgenden Drittanbieter-APIs:

  • Tankerkönig (creativecommons.tankerkoenig.de) — deutsche Spritpreise. Empfängt: Suchkoordinaten, Ihren API-Schlüssel.
  • Prix Carburants (data.economie.gouv.fr) — französische Spritpreise. Empfängt: Suchkoordinaten.
  • Italienische Spritpreis-API (osservaprezzi.mise.gov.it) — italienische Spritpreise. Empfängt: Suchkoordinaten.
  • Spanische Spritpreis-API (sedeaplicaciones.minetur.gob.es) — spanische Spritpreise. Empfängt: Suchkoordinaten.
  • Spritpreisrechner (spritpreisrechner.at) — österreichische Spritpreise. Empfängt: Suchkoordinaten.
  • FuelWatch (fuelwatch.be) — belgische Spritpreise. Empfängt: Suchkoordinaten.
  • data.public.lu — luxemburgische Spritpreise. Empfängt: Suchkoordinaten.
  • OpenChargeMap (api.openchargemap.io) — Ladestationen für E-Fahrzeuge. Empfängt: Suchkoordinaten.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — Geokodierung. Empfängt: Suchtext oder Koordinaten.
  • OpenStreetMap-Kachelserver (tile.openstreetmap.org) — Kartenkacheln. Empfängt: Kachelkoordinaten.
  • Supabase (nur bei aktivem TankSync) — Cloud-Synchronisations-Backend.
  • Sentry (nur bei aktiven Diagnoseberichten) — anonyme Absturz- und Performance-Berichte.

Jeder Anbieter hat eine eigene Datenschutzerklärung. Wir empfehlen, sie zu lesen. Die App tauscht keine Daten zwischen diesen Anbietern aus.

5. Datensicherheit

  • Alle Netzwerkkommunikation läuft über HTTPS (TLS während der Übertragung).
  • API-Schlüssel werden im plattform-eigenen verschlüsselten Speicher abgelegt (Android Keystore / iOS Keychain).
  • Lokale Daten werden auf Ihrem Gerät in Hive gespeichert.
  • Es werden keine Daten an Server gesendet, solange Sie nicht selbst eine Suche starten oder TankSync / Diagnose aktivieren.

6. Ihre Rechte

Sie haben das Recht auf:

  • Auskunft — Sie können alle lokal gespeicherten Daten im Bereich „Speicher“ der App einsehen.
  • Export — Sie können Ihre TankSync-Daten im Bildschirm „Datentransparenz“ als JSON exportieren.
  • Löschung — alle lokalen Daten über Einstellungen → Alle Daten löschen; alle Serverdaten über TankSync → Datentransparenz → Alles löschen.
  • Widerruf — Sie können den Standortzugriff in den Geräteeinstellungen jederzeit widerrufen oder TankSync / Diagnose jederzeit deaktivieren.

7. Datenschutz für Kinder

Die App richtet sich nicht an Kinder unter 13 Jahren. Wir erheben wissentlich keine personenbezogenen Daten von Kindern.
