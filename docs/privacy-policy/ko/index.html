개인정보 처리방침

Sparkilo (iOS의 de.tankstellen.tankstellen, Android의 de.tankstellen.fuelprices) — 최종 업데이트: 2026년 5월 9일

지원 언어: English Deutsch Français Español Italiano Nederlands Português Svenska Suomi Dansk Polski Slovenščina 한국어

1. 개요

Sparkilo(이전 명칭: Fuel Prices Europe & More)는 무료 오픈소스 연료 및 전기차 충전 가격 비교 앱입니다. 로컬 우선, 개인정보를 존중하는 아키텍처로 설계되었습니다. 광고도, 추적 픽셀도, 광고 식별자도 사용하지 않습니다.

2. 사용하는 데이터

2.1 대략적인 위치

위치 권한을 허용하면 앱은 근처 주유소와 충전소를 찾기 위해 대략적인 위치를 읽습니다. 좌표는 검색의 일부로 제3자 API(섹션 4 참조)에 전송됩니다. 위치 정보는 당사가 운영하는 어떤 서버에도 저장되지 않으며 추적이나 프로파일링에 사용되지 않습니다.

2.2 사용자가 제공하는 API 키

사용자가 직접 API 키(예: Tankerkönig)를 제공하면, 해당 키는 암호화된 로컬 저장소(Android Keystore / iOS Keychain)에 저장됩니다. 키는 해당 API 제공자에게만 전송되며, 당사나 제3자에게는 전송되지 않습니다.

2.3 즐겨찾기, 프로필 및 설정

즐겨찾기, 검색 프로필, 연료 환경설정 및 앱 설정은 임베디드 로컬 데이터베이스인 Hive를 통해 기기에 로컬로 저장됩니다.

2.4 TankSync(선택적 클라우드 동기화)

TankSync를 활성화하면 Supabase를 통해 익명 계정이 생성됩니다. 동기화되는 데이터:

  • 익명 사용자 ID(UUID — 이메일 또는 이름 없음)
  • 즐겨찾는 주유소 ID
  • 가격 알림 구성
  • 커뮤니티 가격 보고(주유소 ID, 연료 종류, 가격, 타임스탬프)

TankSync는 선택사항이며 기본적으로 비활성화되어 있습니다. 앱 내 « 데이터 투명성 » 화면에서 모든 서버 측 데이터를 확인, 내보내기 및 삭제할 수 있습니다.

2.5 충돌 및 진단 보고서(옵트인)

설정에서 진단 보고를 활성화하면 익명 충돌 보고서와 성능 추적이 Sentry로 전송됩니다. 개인 식별 정보, 위치 또는 콘텐츠는 포함되지 않습니다. 진단 보고는 기본적으로 꺼져 있습니다.

3. 수집하지 않는 데이터

  • 이름, 이메일 주소 또는 전화번호
  • 금융 또는 결제 정보
  • 건강 또는 피트니스 데이터
  • 연락처, 메시지 또는 통화 기록
  • 사진, 동영상 또는 파일(카메라 및 사진 라이브러리 접근은 사용자가 직접 시작한 영수증 스캔에만 사용됨)
  • 브라우저 기록
  • 기기 광고 식별자

4. 제3자 서비스

앱은 다음 제3자 API와 통신합니다:

  • Tankerkönig(creativecommons.tankerkoenig.de) — 독일 가격. 수신: 검색 좌표, 사용자 API 키.
  • Prix Carburants(data.economie.gouv.fr) — 프랑스 가격. 수신: 검색 좌표.
  • 이탈리아 가격 API(osservaprezzi.mise.gov.it) — 이탈리아 가격. 수신: 검색 좌표.
  • 스페인 가격 API(sedeaplicaciones.minetur.gob.es) — 스페인 가격. 수신: 검색 좌표.
  • Spritpreisrechner(spritpreisrechner.at) — 오스트리아 가격. 수신: 검색 좌표.
  • FuelWatch(fuelwatch.be) — 벨기에 가격. 수신: 검색 좌표.
  • data.public.lu — 룩셈부르크 가격. 수신: 검색 좌표.
  • OpenChargeMap(api.openchargemap.io) — 전기차 충전소. 수신: 검색 좌표.
  • Nominatim / OpenStreetMap(nominatim.openstreetmap.org) — 지오코딩. 수신: 검색 텍스트 또는 좌표.
  • OpenStreetMap 타일 서버(tile.openstreetmap.org) — 지도 타일. 수신: 타일 좌표.
  • Supabase(TankSync 활성화 시에만) — 클라우드 동기화 백엔드.
  • Sentry(진단 보고 활성화 시에만) — 익명 충돌 및 성능 보고서.

각 제공자는 자체 개인정보 처리방침을 보유합니다. 검토하시기 바랍니다. 앱은 이러한 제공자 간에 데이터를 공유하지 않습니다.

5. 데이터 보안

  • 모든 네트워크 통신은 HTTPS(전송 중 TLS 암호화)를 사용합니다.
  • API 키는 플랫폼 기본 암호화 저장소(Android Keystore / iOS Keychain)에 저장됩니다.
  • 로컬 데이터는 Hive를 통해 기기에 저장됩니다.
  • 사용자가 검색을 시작하거나 TankSync / 진단을 활성화하지 않는 한 어떤 서버에도 데이터가 전송되지 않습니다.

6. 사용자의 권리

사용자는 다음 권리를 갖습니다:

  • 접근 — 앱의 저장 공간 섹션에서 로컬에 저장된 모든 데이터를 확인할 수 있습니다.
  • 내보내기 — 데이터 투명성 화면에서 TankSync 데이터를 JSON으로 내보낼 수 있습니다.
  • 삭제 — 설정 → 모든 데이터 삭제로 모든 로컬 데이터를 삭제하고, TankSync → 데이터 투명성 → 모두 삭제로 모든 서버 데이터를 삭제할 수 있습니다.
  • 동의 철회 — 기기 설정에서 위치 권한을 취소하거나 언제든 TankSync / 진단을 비활성화할 수 있습니다.

7. 아동 개인정보

이 앱은 13세 미만 아동을 대상으로 하지 않습니다. 당사는 아동으로부터 개인정보를 고의로 수집하지 않습니다.

8. 정책 변경

당사는 이 정책을 수시로 업데이트할 수 있습니다. 변경사항은 업데이트된 날짜와 함께 이 URL에 게시됩니다. 앱을 계속 사용하는 것은 업데이트된 정책에 대한 동의로 간주됩니다.
