1. Översikt

Sparkilo (tidigare Fuel Prices Europe & More) är en gratis öppen källkods-app för jämförelse av drivmedels- och laddpriser. Den är byggd kring en local-first, integritetsvänlig arkitektur. Inga annonser, inga spårningspixlar, inga annons-ID:n.

2. Data vi använder

2.1 Ungefärlig plats

När du beviljar platsbehörighet läser appen din ungefärliga position för att hitta drivmedels- och laddstationer i närheten. Koordinaterna skickas till tredje parts API:er (se avsnitt 4) som en del av sökningen. Din position lagras inte på någon server vi driver och används aldrig för spårning eller profilering.

2.2 API-nycklar du anger själv

Om du anger en egen API-nyckel (t.ex. Tankerkönig) lagras den lokalt i krypterad lagring (Android Keystore / iOS Keychain). Nyckeln skickas endast till motsvarande API-leverantör, aldrig till oss eller till tredje part.

2.3 Favoriter, profiler och inställningar

Dina favoriter, sökprofiler, drivmedelsval och appinställningar lagras lokalt på din enhet via Hive, en inbyggd lokal databas.

2.4 TankSync (valfri molnsynkronisering)

Om du aktiverar TankSync skapas ett anonymt konto via Supabase. Synkroniserade data:

  • Anonym användar-ID (UUID — ingen e-post eller namn)
  • ID:n för favoritstationer
  • Konfigurationer för prislarm
  • Communityrapporter om priser (stations-ID, drivmedelstyp, pris, tidsstämpel)

TankSync är valfritt och inaktiverat som standard. Du kan visa, exportera och radera all serverdata från skärmen « Datatransparens » i appen.

2.5 Krasch- och diagnosrapporter (opt-in)

Om du aktiverar diagnos i Inställningar skickas anonyma kraschrapporter och prestandaspår till Sentry. Ingen personlig information, plats eller innehåll ingår. Diagnos är avstängd som standard.

3. Data vi INTE samlar in

  • Namn, e-postadress eller telefonnummer
  • Finansiell information eller betalningsuppgifter
  • Hälso- eller träningsdata
  • Kontakter, meddelanden eller samtalsloggar
  • Foton, videor eller filer (åtkomst till kamera och bildbibliotek används endast för kvittoskanning som du själv startar)
  • Webbhistorik
  • Annons-ID:n från enheten

4. Tredjepartstjänster

Appen kommunicerar med följande tredjeparts-API:er:

  • Tankerkönig (creativecommons.tankerkoenig.de) — tyska priser. Tar emot: sökkoordinater, din API-nyckel.
  • Prix Carburants (data.economie.gouv.fr) — franska priser. Tar emot: sökkoordinater.
  • Italiensk pris-API (osservaprezzi.mise.gov.it) — italienska priser. Tar emot: sökkoordinater.
  • Spansk pris-API (sedeaplicaciones.minetur.gob.es) — spanska priser. Tar emot: sökkoordinater.
  • Spritpreisrechner (spritpreisrechner.at) — österrikiska priser. Tar emot: sökkoordinater.
  • FuelWatch (fuelwatch.be) — belgiska priser. Tar emot: sökkoordinater.
  • data.public.lu — luxemburgska priser. Tar emot: sökkoordinater.
  • OpenChargeMap (api.openchargemap.io) — laddstationer. Tar emot: sökkoordinater.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — geokodning. Tar emot: söktext eller koordinater.
  • OpenStreetMap-tile-servrar (tile.openstreetmap.org) — kartrutor. Tar emot: rutkoordinater.
  • Supabase (endast om TankSync är aktivt) — backend för molnsynkronisering.
  • Sentry (endast om diagnos är aktiv) — anonyma krasch- och prestandarapporter.

Varje leverantör har en egen integritetspolicy. Vi rekommenderar att du läser dem. Appen delar inte data mellan dessa leverantörer.

5. Datasäkerhet

  • All nätverkskommunikation använder HTTPS (TLS-kryptering under transport).
  • API-nycklar lagras i plattformens nativa krypterade lagring (Android Keystore / iOS Keychain).
  • Lokal data lagras på din enhet via Hive.
  • Inga data skickas till någon server om du inte själv startar en sökning eller aktiverar TankSync / diagnos.

6. Dina rättigheter

Du har rätt att:

  • Få åtkomst — visa all lokalt lagrad data i appens Lagring-avsnitt.
  • Exportera — exportera dina TankSync-data som JSON från skärmen Datatransparens.
  • Radera — radera all lokal data via Inställningar → Radera all data; radera all serverdata via TankSync → Datatransparens → Radera allt.
  • Återkalla samtycke — återkalla platsbehörigheten i enhetens inställningar, eller stänga av TankSync / diagnos när som helst.

7. Barns integritet

Appen riktar sig inte till barn under 13 år. Vi samlar inte medvetet in personlig information från barn.

8. Ändringar i denna policy

Vi kan uppdatera denna policy från tid till annan. Ändringar publiceras på denna URL med ett uppdaterat datum. Fortsatt användning av appen utgör godkännande av den uppdaterade policyn.
