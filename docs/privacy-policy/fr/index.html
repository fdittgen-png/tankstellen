Politique de confidentialité

Sparkilo (de.tankstellen.tankstellen sous iOS, de.tankstellen.fuelprices sous Android) — Dernière mise à jour : 9 mai 2026

Langues disponibles : English Deutsch Français Español Italiano Nederlands Português Svenska Suomi Dansk Polski Slovenščina 한국어

1. Aperçu

Sparkilo (anciennement Fuel Prices Europe & More) est une application open-source et gratuite de comparaison de prix du carburant et des bornes de recharge. Elle est conçue selon une architecture locale d'abord et respectueuse de la vie privée. Pas de publicité, pas de pixels de suivi, pas d'identifiants publicitaires.

2. Données utilisées

2.1 Position approximative

Lorsque vous accordez l'autorisation de localisation, l'app lit votre position approximative pour trouver les stations à proximité. Les coordonnées sont envoyées aux API tierces (voir section 4) dans le cadre de la requête. Votre position n'est stockée sur aucun serveur que nous exploitons et n'est jamais utilisée à des fins de suivi ou de profilage.

2.2 Clés d'API que vous fournissez

Si vous fournissez votre propre clé d'API (par ex. Tankerkönig), elle est stockée localement dans un coffre chiffré (Android Keystore / iOS Keychain). Elle n'est envoyée qu'au fournisseur d'API correspondant, jamais à nous ni à un tiers.

2.3 Favoris, profils et paramètres

Vos favoris, profils de recherche, préférences de carburant et paramètres sont stockés localement sur votre appareil via Hive, une base de données embarquée.

2.4 TankSync (synchronisation cloud optionnelle)

Si vous activez TankSync, un compte anonyme est créé via Supabase. Les données synchronisées sont :

  • Identifiant utilisateur anonyme (UUID — sans e-mail ni nom)
  • Identifiants des stations favorites
  • Configurations d'alertes de prix
  • Signalements communautaires de prix (ID de station, type de carburant, prix, horodatage)

TankSync est optionnel et désactivé par défaut. Vous pouvez consulter, exporter et supprimer toutes les données côté serveur depuis l'écran « Transparence des données » dans l'app.

2.5 Rapports d'incident et diagnostics (opt-in)

Si vous activez les diagnostics dans les paramètres, des rapports d'incident anonymes et des traces de performance sont envoyés à Sentry. Aucune information personnelle, position ou contenu n'est transmis. Les diagnostics sont désactivés par défaut.

3. Données NON collectées

  • Nom, adresse e-mail ou numéro de téléphone
  • Informations financières ou de paiement
  • Données de santé ou de fitness
  • Contacts, messages ou journaux d'appels
  • Photos, vidéos ou fichiers (l'accès à l'appareil photo et à la photothèque sert uniquement à scanner des reçus à votre initiative)
  • Historique de navigation
  • Identifiants publicitaires de l'appareil

4. Services tiers

L'app communique avec les API tierces suivantes :

  • Tankerkönig (creativecommons.tankerkoenig.de) — prix allemands. Reçoit : coordonnées de recherche, votre clé d'API.
  • Prix Carburants (data.economie.gouv.fr) — prix français. Reçoit : coordonnées de recherche.
  • API italienne des prix (osservaprezzi.mise.gov.it) — prix italiens. Reçoit : coordonnées de recherche.
  • API espagnole des prix (sedeaplicaciones.minetur.gob.es) — prix espagnols. Reçoit : coordonnées de recherche.
  • Spritpreisrechner (spritpreisrechner.at) — prix autrichiens. Reçoit : coordonnées de recherche.
  • FuelWatch (fuelwatch.be) — prix belges. Reçoit : coordonnées de recherche.
  • data.public.lu — prix luxembourgeois. Reçoit : coordonnées de recherche.
  • OpenChargeMap (api.openchargemap.io) — bornes de recharge. Reçoit : coordonnées de recherche.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — géocodage. Reçoit : texte ou coordonnées de recherche.
  • Serveurs de tuiles OpenStreetMap (tile.openstreetmap.org) — fonds de carte. Reçoit : coordonnées de tuile.
  • Supabase (uniquement si TankSync est activé) — backend de synchronisation cloud.
  • Sentry (uniquement si les diagnostics sont activés) — rapports anonymes d'incidents et de performance.

Chaque fournisseur dispose de sa propre politique de confidentialité. Nous vous encourageons à les consulter. L'app ne partage pas de données entre ces fournisseurs.

5. Sécurité des données

  • Toutes les communications réseau utilisent HTTPS (chiffrement TLS en transit).
  • Les clés d'API sont stockées dans le coffre chiffré natif de la plateforme (Android Keystore / iOS Keychain).
  • Les données locales sont stockées sur votre appareil via Hive.
  • Aucune donnée n'est envoyée à un serveur tant que vous n'initiez pas une recherche ou n'activez pas TankSync / les diagnostics.

6. Vos droits

Vous avez le droit de :

  • Accès — consulter toutes les données stockées localement dans la section Stockage de l'app.
  • Export — exporter vos données TankSync au format JSON depuis l'écran Transparence des données.
  • Suppression — supprimer toutes les données locales via Réglages → Tout supprimer ; supprimer toutes les données serveur via TankSync → Transparence des données → Tout supprimer.
  • Retrait du consentement — révoquer l'autorisation de localisation dans les réglages système ou désactiver TankSync / les diagnostics à tout moment.

7. Confidentialité des enfants

L'application ne s'adresse pas aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations personnelles auprès d'enfants.

8. Modifications de cette politique

Cette politique peut être mise à jour de temps à autre. Les modifications seront publiées à cette URL avec une date mise à jour. L'utilisation continue de l'application vaut acceptation de la version mise à jour.
