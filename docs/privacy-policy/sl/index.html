Pravilnik o zasebnosti

Sparkilo (de.tankstellen.tankstellen v iOS-u, de.tankstellen.fuelprices v Androidu) — Zadnja posodobitev: 9. maj 2026

Razpoložljivi jeziki: English Deutsch Français Español Italiano Nederlands Português Svenska Suomi Dansk Polski Slovenščina 한국어

1. Pregled

Sparkilo (prej Fuel Prices Europe & More) je brezplačna odprtokodna aplikacija za primerjavo cen goriv in polnilnih postaj za električna vozila. Zgrajena je po local-first, zasebnost spoštujočem načelu. Brez oglasov, brez sledilnih pikslov, brez oglaševalskih ID-jev.

2. Podatki, ki jih uporabljamo

2.1 Približna lokacija

Ko dovolite dostop do lokacije, aplikacija prebere vaš približni položaj, da poišče bližnje črpalke in polnilne postaje. Koordinate se kot del poizvedbe pošljejo API-jem tretjih oseb (glejte razdelek 4). Vaša lokacija se ne shrani na noben naš strežnik in se nikoli ne uporablja za sledenje ali profiliranje.

2.2 API ključi, ki jih posredujete sami

Če posredujete svoj API ključ (npr. Tankerkönig), je shranjen lokalno v šifriranem shranjevalniku (Android Keystore / iOS Keychain). Ključ se pošlje samo ustreznemu ponudniku API-ja, nikoli nam ali tretjim osebam.

2.3 Priljubljene, profili in nastavitve

Vaše priljubljene, iskalni profili, nastavitve goriv in nastavitve aplikacije se shranjujejo lokalno na vašo napravo v zbirko Hive.

2.4 TankSync (neobvezna sinhronizacija v oblaku)

Če omogočite TankSync, se prek storitve Supabase ustvari anonimni račun. Sinhronizirani podatki:

  • Anonimni ID uporabnika (UUID — brez e-pošte ali imena)
  • ID-ji priljubljenih črpalk
  • Nastavitve cenovnih opozoril
  • Skupnostni podatki o cenah (ID črpalke, vrsta goriva, cena, časovni žig)

TankSync je neobvezen in privzeto izklopljen. Vse podatke na strežniku si lahko ogledate, izvozite in izbrišete iz zaslona « Preglednost podatkov » znotraj aplikacije.

2.5 Poročila o sesutjih in diagnostika (po izbiri)

Če v Nastavitvah omogočite diagnostiko, se v Sentry pošljejo anonimna poročila o sesutjih in sledi delovanja. Vključenih ni nobenih osebnih podatkov, lokacije ali vsebine. Diagnostika je privzeto izklopljena.

3. Podatki, ki jih NE zbiramo

  • Ime, e-poštni naslov ali telefonsko številko
  • Finančne ali plačilne podatke
  • Zdravstvene ali fitnes podatke
  • Stike, sporočila ali dnevnike klicev
  • Fotografije, video posnetke ali datoteke (dostop do kamere in galerije se uporablja le za skeniranje računov, ki ga sami sprožite)
  • Zgodovino brskanja
  • Oglaševalske identifikatorje naprave

4. Storitve tretjih oseb

Aplikacija komunicira z naslednjimi API-ji tretjih oseb:

  • Tankerkönig (creativecommons.tankerkoenig.de) — nemške cene. Prejme: koordinate iskanja, vaš API ključ.
  • Prix Carburants (data.economie.gouv.fr) — francoske cene. Prejme: koordinate iskanja.
  • Italijanski API cen (osservaprezzi.mise.gov.it) — italijanske cene. Prejme: koordinate iskanja.
  • Španski API cen (sedeaplicaciones.minetur.gob.es) — španske cene. Prejme: koordinate iskanja.
  • Spritpreisrechner (spritpreisrechner.at) — avstrijske cene. Prejme: koordinate iskanja.
  • FuelWatch (fuelwatch.be) — belgijske cene. Prejme: koordinate iskanja.
  • data.public.lu — luksemburške cene. Prejme: koordinate iskanja.
  • OpenChargeMap (api.openchargemap.io) — polnilne postaje. Prejme: koordinate iskanja.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — geokodiranje. Prejme: iskalno besedilo ali koordinate.
  • Strežniki ploščic OpenStreetMap (tile.openstreetmap.org) — zemljevidne ploščice. Prejme: koordinate ploščice.
  • Supabase (samo če je TankSync omogočen) — zaledje za sinhronizacijo v oblaku.
  • Sentry (samo če je diagnostika omogočena) — anonimna poročila o sesutjih in delovanju.

Vsak ponudnik ima svoj pravilnik o zasebnosti. Priporočamo, da jih preberete. Aplikacija med temi ponudniki ne deli podatkov.

5. Varnost podatkov

  • Vsa omrežna komunikacija poteka prek HTTPS (TLS šifriranje med prenosom).
  • API ključi se shranjujejo v platformi lastnem šifriranem shranjevalniku (Android Keystore / iOS Keychain).
  • Lokalni podatki se shranjujejo na vaši napravi prek Hive.
  • Nobeni podatki se ne pošiljajo na strežnik, razen če sami sprožite iskanje ali omogočite TankSync / diagnostiko.

6. Vaše pravice

Imate pravico do:

  • Vpogleda — vse lokalno shranjene podatke si lahko ogledate v razdelku Shramba znotraj aplikacije.
  • Izvoza — podatke TankSync lahko izvozite v JSON s zaslona Preglednost podatkov.
  • Izbrisa — vse lokalne podatke izbrišete prek Nastavitve → Izbriši vse podatke; vse strežniške podatke prek TankSync → Preglednost podatkov → Izbriši vse.
  • Preklica privolitve — preklicati dovoljenje za lokacijo v sistemskih nastavitvah ali kadar koli onemogočiti TankSync / diagnostiko.

7. Zasebnost otrok

Aplikacija ni namenjena otrokom, mlajšim od 13 let. Zavestno ne zbiramo osebnih podatkov otrok.

8. Spremembe tega pravilnika

Pravilnik lahko občasno posodobimo. Spremembe bomo objavili na tem URL-ju z ažurnim datumom. Z nadaljnjo uporabo aplikacije se strinjate s posodobljenim pravilnikom.
