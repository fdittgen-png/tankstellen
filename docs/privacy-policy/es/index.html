Política de privacidad

Sparkilo (de.tankstellen.tankstellen en iOS, de.tankstellen.fuelprices en Android) — Última actualización: 9 de mayo de 2026

Idiomas disponibles: English Deutsch Français Español Italiano Nederlands Português Svenska Suomi Dansk Polski Slovenščina 한국어

1. Resumen

Sparkilo (antes Fuel Prices Europe & More) es una aplicación gratuita y de código abierto para comparar precios de carburante y de puntos de recarga eléctrica. Está diseñada con una arquitectura local-first y respetuosa con la privacidad. Sin anuncios, sin píxeles de seguimiento, sin identificadores publicitarios.

2. Datos que utilizamos

2.1 Ubicación aproximada

Cuando concede el permiso de ubicación, la app lee su posición aproximada para encontrar gasolineras y puntos de recarga cercanos. Las coordenadas se envían a APIs de terceros (ver sección 4) como parte de la consulta. Su ubicación no se almacena en ningún servidor que operemos y nunca se utiliza para seguimiento o perfilado.

2.2 Claves de API que usted proporciona

Si proporciona su propia clave de API (por ejemplo de Tankerkönig), se almacena localmente en almacenamiento cifrado (Android Keystore / iOS Keychain). La clave se envía únicamente al proveedor correspondiente, nunca a nosotros ni a terceros.

2.3 Favoritos, perfiles y ajustes

Sus favoritos, perfiles de búsqueda, preferencias de combustible y ajustes se almacenan localmente en su dispositivo mediante Hive, una base de datos local embebida.

2.4 TankSync (sincronización en la nube opcional)

Si activa TankSync, se crea una cuenta anónima en Supabase. Los datos sincronizados son:

  • Identificador anónimo de usuario (UUID — sin correo ni nombre)
  • IDs de gasolineras favoritas
  • Configuraciones de alertas de precio
  • Reportes comunitarios de precios (ID de gasolinera, tipo de combustible, precio, marca temporal)

TankSync es opcional y está desactivado por defecto. Puede ver, exportar y eliminar todos los datos del servidor desde la pantalla « Transparencia de datos » dentro de la app.

2.5 Informes de error y diagnóstico (opt-in)

Si activa los diagnósticos en Ajustes, se envían informes anónimos de fallos y trazas de rendimiento a Sentry. No se incluye ninguna información personal, ubicación ni contenido. Los diagnósticos están desactivados por defecto.

3. Datos que NO recogemos

  • Nombre, dirección de correo o número de teléfono
  • Información financiera o de pago
  • Datos de salud o fitness
  • Contactos, mensajes o registros de llamadas
  • Fotos, vídeos o archivos (el acceso a la cámara y a la galería se usa solo para escanear tickets cuando usted lo inicia)
  • Historial de navegación
  • Identificadores publicitarios del dispositivo

4. Servicios de terceros

La app se comunica con las siguientes APIs de terceros:

  • Tankerkönig (creativecommons.tankerkoenig.de) — precios alemanes. Recibe: coordenadas de búsqueda, su clave de API.
  • Prix Carburants (data.economie.gouv.fr) — precios franceses. Recibe: coordenadas de búsqueda.
  • API italiana de precios (osservaprezzi.mise.gov.it) — precios italianos. Recibe: coordenadas de búsqueda.
  • API española de precios (sedeaplicaciones.minetur.gob.es) — precios españoles. Recibe: coordenadas de búsqueda.
  • Spritpreisrechner (spritpreisrechner.at) — precios austríacos. Recibe: coordenadas de búsqueda.
  • FuelWatch (fuelwatch.be) — precios belgas. Recibe: coordenadas de búsqueda.
  • data.public.lu — precios luxemburgueses. Recibe: coordenadas de búsqueda.
  • OpenChargeMap (api.openchargemap.io) — puntos de recarga eléctrica. Recibe: coordenadas de búsqueda.
  • Nominatim / OpenStreetMap (nominatim.openstreetmap.org) — geocodificación. Recibe: texto o coordenadas de búsqueda.
  • Servidores de tiles OpenStreetMap (tile.openstreetmap.org) — tiles del mapa. Recibe: coordenadas del tile.
  • Supabase (solo si TankSync está activado) — backend de sincronización en la nube.
  • Sentry (solo si los diagnósticos están activados) — informes anónimos de fallos y rendimiento.

Cada proveedor tiene su propia política de privacidad. Le recomendamos revisarlas. La app no comparte datos entre estos proveedores.

5. Seguridad de los datos

  • Toda la comunicación de red usa HTTPS (cifrado TLS en tránsito).
  • Las claves de API se almacenan en el almacenamiento cifrado nativo de la plataforma (Android Keystore / iOS Keychain).
  • Los datos locales se almacenan en su dispositivo mediante Hive.
  • No se envían datos a ningún servidor a menos que inicie una búsqueda o active TankSync / los diagnósticos.

6. Sus derechos

Tiene derecho a:

  • Acceso — ver todos los datos almacenados localmente en la sección Almacenamiento de la app.
  • Exportación — exportar sus datos de TankSync como JSON desde la pantalla Transparencia de datos.
  • Supresión — eliminar todos los datos locales mediante Ajustes → Eliminar todos los datos; eliminar todos los datos del servidor mediante TankSync → Transparencia de datos → Eliminar todo.
  • Retirar el consentimiento — revocar el permiso de ubicación en los ajustes del sistema o desactivar TankSync / los diagnósticos en cualquier momento.

7. Privacidad de los menores

La aplicación no está dirigida a menores de 13 años. No recopilamos a sabiendas información personal de menores.

8. Cambios en esta política

Podemos actualizar esta política de vez en cuando. Los cambios se publicarán en esta URL con una fecha actualizada. El uso continuado de la aplicación constituye aceptación de la política actualizada.
