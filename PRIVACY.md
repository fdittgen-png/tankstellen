## Summary

This app collects minimal data and has no server. All data stays on your device.
No analytics, no tracking, no advertising.

## Data Collection

### Data sent to third-party services

| Data | Sent To | When | Purpose |
|------|---------|------|---------|
| GPS coordinates (latitude, longitude) | Fuel price API (Prix-Carburants, Tankerkoenig) | When you search by GPS | To find nearby fuel stations |
| Postal code | Nominatim (OpenStreetMap) | When you search by postal code | To convert postal code to coordinates |
| Map viewport | OpenStreetMap tile servers | When you view the map | To load map tiles |
| API key (if configured) | Tankerkoenig API | With every API request | Authentication |

### Data stored on your device only

- Profile settings (name, preferred fuel, search radius)
- Favorite stations
- Search cache (automatically expires)
- Error traces (if enabled, for debugging)
- Language and country preferences

### Data NOT collected

- Search history is not stored permanently
- No user accounts are required
- No personal information beyond what you voluntarily enter
- No usage analytics or telemetry
- No advertising identifiers

## Error Trace Upload (Optional)

If you enable error trace upload in Settings, technical diagnostic data
(device type, app version, error details) may be sent to a server URL
you configure. This is disabled by default and entirely optional.
Error traces never contain search history, favorites, or personal data.

## Your Rights (GDPR / DSGVO)

Since this app stores data only on your device and has no server-side
user accounts, most GDPR rights are exercised directly:

- **Right to access:** View all stored data in the Profile screen
- **Right to erasure:** Use "Delete all data" in Profile > Storage & Cache
- **Right to data portability:** Error traces can be exported as JSON
- **Right to withdraw consent:** Revoke location permission in system settings
